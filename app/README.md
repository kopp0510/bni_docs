# 商脈會 App 客端規劃

## 專案概述

商脈會 App 客端是一個商業會員互助平台，提供會員交流、引薦服務、商學講堂、點數管理等功能。

---

## 技術棧

| 項目 | 技術選型 |
|------|----------|
| 框架 | React Native / Flutter |
| 狀態管理 | Redux / Provider |
| 網路請求 | Axios / Dio |
| 即時通訊 | Socket.io |
| 推播通知 | Firebase Cloud Messaging |
| 本地儲存 | AsyncStorage / SharedPreferences |

## 會員等級系統

| 等級 | 顏色 | 說明 |
|------|------|------|
| 金質 | 金色 | 最高等級會員 |
| 優質 | 綠色 | 中階等級會員 |
| 一般 | 黑色 | 基礎等級會員 |

等級根據以下指標提升：
- 會員交流次數
- 商學講堂參與
- 優質引薦次數
- 推薦入會次數
- 活動參與次數

## UI 設計規範

### 主色調

| 用途 | 顏色 |
|------|------|
| 主色 | 淺黃色（會員卡背景） |
| 強調色 | 綠色（按鈕、標籤） |
| 背景色 | 淺藍色（頁面背景） |
| 文字色 | 深灰/黑色 |

### 會員卡片樣式

- 圓角矩形
- 淺黃色背景
- 包含：頭像、姓名、等級標籤、商會、產業別
- 聯絡資訊：電話、Email
- 統計數據：會員交流、商學講堂、優質引薦、推薦入會、活動參與

### 廣告輪播

- 比例：16:9 或 2:1
- 自動輪播
- 指示點顯示

### 跑馬燈

- 單行滾動文字
- 淺色背景
- 喇叭圖示前綴
