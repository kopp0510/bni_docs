## 報名流程

```
課程列表 → 點擊「立即報名」→ 課程詳情確認 → 送出報名 → 報名成功
```

### 報名檢查

| 檢查項目 | 說明 |
|----------|------|
| 點數餘額 | 確認點數足夠 |
| 報名狀態 | 確認未重複報名 |
| 報名人數 | 確認未額滿 |
| 報名時間 | 確認在報名期間內 |

### 報名成功

- 顯示成功訊息
- 扣除消耗點數
- 加入「我的課程」
- 發送報名通知
- 活動前一天發送提醒

## 簽到功能

課程開始前後一定時間內，可進行簽到。

### 簽到方式

| 方式 | 說明 |
|------|------|
| QR Code 簽到 | 掃描現場 QR Code |
| 定位簽到 | 確認在活動地點範圍內 |
| 手動簽到 | 工作人員手動確認 |

---

## API 呼叫

| 功能 | API | 說明 |
|------|-----|------|
| 課程列表 | GET /api/v1/lectures | 取得課程列表 |
| 課程詳情 | GET /api/v1/lectures/:id | 取得課程詳情 |
| 課程類型 | GET /api/v1/lectures/types | 取得課程類型 |
| 報名課程 | POST /api/v1/lectures/:id/register | 報名課程 |
| 取消報名 | DELETE /api/v1/lectures/:id/register | 取消報名 |
| 我的課程 | GET /api/v1/lectures/mine | 取得已報名課程 |
| 簽到 | POST /api/v1/lectures/:id/check-in | 課程簽到 |

## 錯誤處理

| 錯誤代碼 | 說明 | 處理方式 |
|----------|------|----------|
| LEC_001 | 課程不存在 | 顯示錯誤訊息 |
| LEC_002 | 非報名期間 | 顯示報名時間 |
| LEC_003 | 報名已額滿 | 顯示已額滿 |
| LEC_004 | 點數不足 | 提示充值點數 |
| LEC_005 | 已報名此課程 | 顯示已報名 |
| LEC_006 | 簽到時間未到 | 顯示簽到時間 |
| LEC_007 | 不在簽到範圍 | 提示靠近簽到地點 |
