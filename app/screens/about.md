# 認識愛心 (About)

## 頁面概述

認識愛心頁面介紹商脈會組織資訊，包含本會宗旨、幹部介紹、各地分會等內容。

## 功能項目

### 1. 本會宗旨

顯示商脈會的組織宗旨與理念。

| 內容 | 說明 |
|------|------|
| 標題 | 本會宗旨 |
| 內容 | 組織理念、使命、願景 |
| 格式 | 圖文排版 |

---

### 2. 本會幹部

顯示組織幹部資訊。

**幹部列表：**

| 欄位 | 說明 |
|------|------|
| 頭像 | 幹部照片 |
| 姓名 | 幹部姓名 |
| 職稱 | 會長/副會長/秘書長 等 |
| 簡介 | 個人簡介 |

---

### 3. 各地分會

顯示全台各地分會資訊。

**分會列表：**

| 欄位 | 說明 |
|------|------|
| 分會名稱 | 台北總會、台中總會 等 |
| 負責人 | 分會長姓名 |
| 聯絡電話 | 分會電話 |
| 地址 | 分會地址 |
| 成立時間 | 分會成立日期 |

**地圖顯示：**
- 顯示各分會位置
- 點擊可導航

---

### 4. 加入愛心

引導使用者加入商脈會。

| 內容 | 說明 |
|------|------|
| 說明 | 入會說明、資格條件 |
| 流程 | 入會流程步驟 |
| 費用 | 會費說明 |
| 申請 | 跳轉至註冊頁面 |

---

### 5. 粉專、YT

連結至社群媒體。

| 平台 | 說明 |
|------|------|
| Facebook 粉專 | 開啟 Facebook 頁面 |
| YouTube 頻道 | 開啟 YouTube 頻道 |
| Instagram | 開啟 IG 頁面 |
| LINE 官方帳號 | 加入 LINE 好友 |

---

## API 呼叫

| 功能 | API | 說明 |
|------|-----|------|
| 組織資訊 | GET /api/v1/cms/about | 取得組織介紹 |
| 幹部列表 | GET /api/v1/cms/executives | 取得幹部資訊 |
| 分會列表 | GET /api/v1/organizations | 取得分會列表 |
