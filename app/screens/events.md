## 報名流程

```
活動列表 → 點擊「立即報名」→ 活動詳情確認 → 送出報名 → 報名成功
```

### 報名確認

| 檢查項目 | 說明 |
|----------|------|
| 點數餘額 | 確認點數足夠 |
| 報名狀態 | 確認未重複報名 |
| 報名人數 | 確認未額滿 |
| 報名時間 | 確認在報名期間內 |

### 報名成功

- 顯示成功訊息
- 扣除消耗點數
- 加入「我的活動」
- 發送報名通知

---

## 我的活動

顯示已報名的活動列表。

| Tab | 說明 |
|-----|------|
| 即將開始 | 尚未開始的活動 |
| 已結束 | 已完成的活動 |
| 已取消 | 取消報名的活動 |

### 活動狀態

| 狀態 | 說明 |
|------|------|
| REGISTERED | 已報名 |
| CONFIRMED | 已確認 |
| CHECKED_IN | 已簽到 |
| COMPLETED | 已完成 |
| CANCELLED | 已取消 |
| NO_SHOW | 未出席 |

## API 呼叫

| 功能 | API | 說明 |
|------|-----|------|
| 活動列表 | GET /api/v1/events | 取得活動列表 |
| 活動詳情 | GET /api/v1/events/:id | 取得活動詳情 |
| 活動類型 | GET /api/v1/events/types | 取得活動類型 |
| 報名活動 | POST /api/v1/events/:id/register | 報名活動 |
| 取消報名 | DELETE /api/v1/events/:id/register | 取消報名 |
| 我的活動 | GET /api/v1/events/mine | 取得已報名活動 |

---

## 錯誤處理

| 錯誤代碼 | 說明 | 處理方式 |
|----------|------|----------|
| EVT_002 | 活動非報名中 | 顯示活動狀態 |
| EVT_003 | 報名人數已額滿 | 顯示已額滿 |
| EVT_004 | 已超過報名時間 | 顯示報名已截止 |
| EVT_005 | 點數不足 | 提示充值點數 |
| EVT_006 | 已報名此活動 | 顯示已報名 |
