# 好康優惠 (Deals)

## 頁面概述

好康優惠是 Tab 2 的主要頁面，顯示各種優惠活動、商家特惠、限時折扣等資訊。

## API 呼叫

| 功能 | API | 說明 |
|------|-----|------|
| 優惠列表 | GET /api/v1/deals | 取得優惠列表 |
| 優惠詳情 | GET /api/v1/deals/:id | 取得優惠詳情 |
| 優惠類型 | GET /api/v1/deals/types | 取得優惠類型 |
| 收藏優惠 | POST /api/v1/deals/:id/favorite | 收藏優惠 |
| 取消收藏 | DELETE /api/v1/deals/:id/favorite | 取消收藏 |
| 我的優惠 | GET /api/v1/deals/mine | 取得我的優惠 |
| 輪播廣告 | GET /api/v1/deals/banners | 取得輪播廣告 |

## 錯誤處理

| 錯誤代碼 | 說明 | 處理方式 |
|----------|------|----------|
| DEL_001 | 優惠不存在 | 顯示錯誤訊息 |
| DEL_002 | 優惠已結束 | 顯示已結束訊息 |
| DEL_003 | 不符合使用條件 | 顯示條件說明 |
| DEL_004 | 已達使用上限 | 顯示已使用訊息 |
