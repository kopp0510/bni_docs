### 捐贈表單

| 欄位 | 類型 | 必填 | 說明 |
|------|------|------|------|
| 捐贈金額 | 數字輸入 | 是 | 捐贈點數 |
| 快速選擇 | 按鈕 | 否 | 預設金額 |
| 捐贈留言 | 文字輸入 | 否 | 留言訊息 |
| 匿名捐贈 | 勾選 | 否 | 是否匿名 |

## API 呼叫

| 功能 | API | 說明 |
|------|-----|------|
| 團體列表 | GET /api/v1/charities | 取得公益團體列表 |
| 團體詳情 | GET /api/v1/charities/:id | 取得團體詳情 |
| 團體類型 | GET /api/v1/charities/types | 取得公益類型 |
| 捐贈點數 | POST /api/v1/charities/:id/donate | 執行捐贈 |
| 捐贈紀錄 | GET /api/v1/charities/donations | 取得捐贈紀錄 |
| 捐贈收據 | GET /api/v1/donations/:id/receipt | 取得捐贈收據 |

## 錯誤處理

| 錯誤代碼 | 說明 | 處理方式 |
|----------|------|----------|
| CHR_001 | 團體不存在 | 顯示錯誤訊息 |
| CHR_002 | 募款已結束 | 顯示已結束訊息 |
| CHR_003 | 點數不足 | 提示充值或減少金額 |
| CHR_004 | 捐贈金額無效 | 顯示最低/最高限制 |
