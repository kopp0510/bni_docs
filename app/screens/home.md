# 首頁 (Home)

## 頁面概述

## 功能元件

### 1. 會員資訊卡

| 欄位 | 說明 |
|------|------|
| 頭像 | 會員大頭照 |
| 姓名 | 會員姓名 |
| 等級標籤 | 金質/優質/一般（顏色區分） |
| 會員類型 | 一般會員/企業會員/公益商家 |
| 所屬商會 | 台中總會 等 |
| 會員編號 | 20250001 |
| 會期 | 2025/2/1-2026/1/31 |
| 年資 | 入會至今時間 |

### 2. 點數顯示區

| 項目 | 說明 |
|------|------|
| 愛心點數 | 目前可用點數（黃底顯眼） |
| 到期日 | 點數有效期限 |
| 獎勵點數 | 額外獎勵點數 |

### 3. 廣告輪播

| 項目 | 說明 |
|------|------|
| 圖片比例 | 16:9 |
| 輪播間隔 | 5 秒 |
| 指示點 | 顯示目前位置 |
| 點擊行為 | 跳轉至指定頁面或網址 |

### 4. 跑馬燈公告

| 項目 | 說明 |
|------|------|
| 樣式 | 淺藍色背景，單行滾動 |
| 圖示 | 喇叭 📢 |
| 內容來源 | 後台 CMS 設定 |

### 5. 六宮格功能入口

| 功能 | 圖示 | 跳轉頁面 |
|------|------|----------|
| 認識愛心 | 💬❤️ | 組織介紹頁 |
| 公益商家 | 📍❤️ | 公益商家列表 |
| 會員專區 | 👥 | 會員專區主頁 |
| 最新活動 | 📣 | 活動列表頁 |
| 公益團體 | ❤️ | 公益團體/捐贈頁 |
| 使用點數 | 💕Pay | 點數功能主頁 |

## API 呼叫

| 時機 | API | 說明 |
|------|-----|------|
| 頁面載入 | GET /api/v1/members/me | 取得會員資訊 |
| 頁面載入 | GET /api/v1/cms/banners | 取得廣告輪播 |
| 頁面載入 | GET /api/v1/cms/announcements | 取得跑馬燈公告 |
| 下拉刷新 | 同上 | 重新取得所有資料 |

---

## 使用者互動

| 操作 | 行為 |
|------|------|
| 點擊頭像 | 跳轉至個人資料頁 |
| 點擊點數區 | 跳轉至點數紀錄頁 |
| 滑動廣告 | 手動切換廣告 |
| 點擊廣告 | 跳轉至指定頁面 |
| 點擊功能入口 | 跳轉至對應功能頁 |
| 下拉 | 重新整理頁面 |
