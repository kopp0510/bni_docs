# 公益商家 (Public Merchant)

## 頁面概述

公益商家列表顯示所有加入公益愛心 Pay 的商家，會員可瀏覽商家資訊、使用點數消費。

## 點數付款流程

```
商家詳情 → 點擊「點數付款」→ 掃描商家 QR Code / 輸入金額 → 確認付款 → 完成
```

或透過：

```
使用點數 → 掃碼付款 → 掃描商家 QR Code → 輸入金額 → 確認付款 → 完成
```

## 評論功能

### 發表評論

| 欄位 | 說明 |
|------|------|
| 評分 | 1-5 星 |
| 評論內容 | 文字評論 |
| 照片 | 上傳照片（選填） |

### 評論規則

- 須消費過才能評論
- 每次消費可評論一次
- 評論可編輯/刪除

## API 呼叫

| 功能 | API | 說明 |
|------|-----|------|
| 商家列表 | GET /api/v1/shops | 取得商家列表 |
| 商家詳情 | GET /api/v1/shops/:id | 取得商家詳情 |
| 商家類別 | GET /api/v1/shops/categories | 取得商家類別 |
| 附近商家 | GET /api/v1/shops/nearby | 取得附近商家 |
| 商家評論 | GET /api/v1/shops/:id/reviews | 取得商家評論 |
| 發表評論 | POST /api/v1/shops/:id/reviews | 發表評論 |
| 收藏商家 | POST /api/v1/shops/:id/favorite | 收藏商家 |

## 錯誤處理

| 錯誤代碼 | 說明 | 處理方式 |
|----------|------|----------|
| SHP_001 | 商家不存在 | 顯示錯誤訊息 |
| SHP_002 | 商家已停業 | 顯示停業訊息 |
| SHP_003 | 定位服務未開啟 | 提示開啟定位 |
| SHP_004 | 無法發表評論 | 顯示評論條件 |
