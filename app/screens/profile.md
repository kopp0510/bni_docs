# 關於我 (Profile)

## 頁面概述

關於我是 Tab 5 的主要頁面，顯示會員個人資料、引薦紀錄、交流紀錄及各項設定。

| 欄位 | 說明 |

### 可編輯欄位

| 欄位 | 類型 | 說明 |
|------|------|------|
| 頭像 | 圖片上傳 | 更換頭像照片 |
| 姓名 | 文字 | 真實姓名 |
| Email | 電子郵件 | 電子信箱 |
| 生日 | 日期選擇 | 出生日期 |
| 性別 | 單選 | 男/女/其他 |
| 公司名稱 | 文字 | 公司/店家名稱 |
| 產業別 | 下拉選擇 | 所屬產業 |
| 自我介紹 | 多行文字 | 個人簡介 |

### 不可編輯欄位

| 欄位 | 說明 |
|------|------|
| 手機號碼 | 需透過客服變更 |
| 所屬商會 | 需透過客服變更 |
| 會員等級 | 系統自動計算 |


### 修改密碼

| 欄位 | 說明 |
|------|------|
| 目前密碼 | 驗證身份 |
| 新密碼 | 至少 8 位含英數 |
| 確認密碼 | 再次輸入新密碼 |

### 隱私設定

| 設定 | 說明 |
|------|------|
| 允許搜尋 | 其他會員是否能搜尋到我 |
| 顯示聯絡資訊 | 是否顯示手機/Email |
| 接受交流預約 | 是否開放交流預約功能 |

## 登出

點擊登出後：
1. 清除本地 Token
2. 清除快取資料
3. 返回登入頁面

---

## API 呼叫

| 功能 | API | 說明 |
|------|-----|------|
| 個人資料 | GET /api/v1/members/me | 取得個人資料 |
| 更新資料 | PUT /api/v1/members/me | 更新個人資料 |
| 上傳頭像 | POST /api/v1/members/me/avatar | 上傳頭像 |
| 修改密碼 | PUT /api/v1/auth/change-password | 修改密碼 |
| 我的引薦 | GET /api/v1/referrals/mine | 取得我的引薦 |
| 我的交流 | GET /api/v1/meetings/mine | 取得我的交流 |
| 我的活動 | GET /api/v1/events/mine | 取得我的活動 |
| 我的課程 | GET /api/v1/lectures/mine | 取得我的課程 |
| 隱私設定 | PUT /api/v1/members/me/privacy | 更新隱私設定 |
| 意見回饋 | POST /api/v1/feedback | 送出回饋 |
| 登出 | POST /api/v1/auth/logout | 登出 |

## 錯誤處理

| 錯誤代碼 | 說明 | 處理方式 |
|----------|------|----------|
| PRF_001 | 密碼錯誤 | 顯示錯誤訊息 |
| PRF_002 | 新密碼格式不符 | 顯示密碼規則 |
| PRF_003 | 頭像上傳失敗 | 提示重試 |
| PRF_004 | 資料更新失敗 | 顯示錯誤訊息 |
