# 會員專區 (Member Zone)

## 頁面概述

## 六宮格功能

### 1. 會員搜尋

搜尋並查看其他會員的詳細資料。

**功能流程：**
```
會員列表 → 點擊會員 → 會員詳情頁
         ↓
    搜尋/篩選
```

**會員列表頁：**

| 元件 | 說明 |
|------|------|
| 商會篩選 | 下拉選擇商會 |
| 搜尋框 | 全部搜尋（姓名、產業） |
| 會員卡片 | 頭像、姓名、等級、產業別、電話、Email |
| 操作按鈕 | 加入好友、發起交流 |

**會員詳情頁：**

| 區塊 | 內容 |
|------|------|
| 基本資訊 | 頭像、姓名、等級、產業別、電話、Email |
| 統計數據 | 會員交流、商學講堂、優質引薦、推薦入會、活動參與 |
| 自我介紹 | 文字介紹區塊 |
| 動態名片 | 詳細個人/企業介紹 |
| 操作按鈕 | 加入好友、發起交流 |

**動態名片欄位：**

| 欄位 | 說明 |
|------|------|
| 姓名 | 會員姓名 |
| 產業別 | 所屬產業 |
| 公司名稱 | 公司/店家名稱 |
| 實際案例 | 成功案例描述 |
| 十大客戶 | 主要客戶列表 |
| 核心產品及服務 | 主要服務項目 |
| 你與競爭對手有什麼不同 | 差異化優勢 |
| 你的目標市場與主力客群 | 目標客群 |
| 你所需引薦的相關行業 | 期望合作對象 |

---

### 2. 會員交流

與其他會員預約 1 對 1 交流見面。

**功能流程：**
```
會員列表 → 點擊「發起交流」→ 填寫交流表單 → 送出
```

**會員列表頁：**

| 元件 | 說明 |
|------|------|
| 商會篩選 | 下拉選擇商會 |
| 搜尋框 | 全部搜尋 |
| 會員卡片 | 含「發起交流」按鈕 |

**發起交流表單：**

| 欄位 | 類型 | 必填 | 說明 |
|------|------|------|------|
| 會面人 | 搜尋選擇 | 是 | 選擇要交流的會員 |
| 發起人 | 自動帶入 | - | 目前登入會員 |
| 地點 | 文字輸入 | 是 | 見面地點 |
| 日期 | 日期選擇 | 是 | 見面日期 |
| 會談主題 | 文字輸入 | 是 | 交流主題 |

---

### 3. 商學講堂

瀏覽並報名商學課程。

**功能流程：**
```
課程列表 → 點擊「立即報名」→ 確認報名資訊 → 送出
```

**課程列表頁：**

| 元件 | 說明 |
|------|------|
| 商會篩選 | 下拉選擇商會 |
| 課程卡片 | 封面、標題、講師、日期時間、點數、分數 |
| 報名按鈕 | 立即報名 |

**課程卡片資訊：**

| 項目 | 說明 |
|------|------|
| 封面圖 | 課程封面 |
| 講師資訊 | 商學講師-XXX |
| 課程標題 | 課程名稱 |
| 日期 | 📅 2025/01/01 |
| 時間 | 🕐 13:30-14:30 |
| 點數消耗 | 500點 |
| 獲得分數 | 分數+2 |

**報名確認頁：**

| 項目 | 說明 |
|------|------|
| 會員資訊 | 報名者資訊卡 |
| 課程資訊 | 課程詳情 |
| 地點資訊 | 公司名稱、地址 |
| 送出按鈕 | 確認報名 |
| 取消按鈕 | 取消報名 |

### 4. PT 專區

查看各組別會員與團隊介紹。

**功能流程：**
```
組別列表 → 點擊組別 → 組別會員列表 → 團隊介紹 / 服務引薦
```

**組別列表頁：**

| 元件 | 說明 |
|------|------|
| 商會篩選 | 下拉選擇商會 |
| 會員卡片 | 組別、角色（組長）、姓名、等級、產業、電話、Email |
| 操作按鈕 | 加入好友、團隊介紹、服務引薦 |

**組別詳情頁：**

| 區塊 | 說明 |
|------|------|
| 組別名稱 | 大健康組 (台中總會) |
| 成員列表 | 組長、組員卡片 |
| 操作按鈕 | 加入好友、發起交流 |

---

### 5. 優質引薦

查看和提供引薦服務。

**功能流程：**
```
引薦列表（提供/收到）→ 點擊「服務引薦」→ 填寫引薦表單 → 送出
```

**引薦列表頁：**

| Tab | 說明 |
|-----|------|
| 提供的引薦 | 我提供給其他會員的引薦 |
| 收到的引薦 | 其他會員提供給我的引薦 |

**引薦卡片：**

| 項目 | 說明 |
|------|------|
| 會員資訊 | 頭像、姓名、等級、產業 |
| 引薦內容 | 引薦描述 |
| 更多內容 | 展開查看完整內容 |

**服務引薦表單：**

| 欄位 | 類型 | 必填 | 說明 |
|------|------|------|------|
| 引薦對象 | Tab 切換 | 是 | 會員 / 非會員 |
| 聯繫方式 | 複選框 | 否 | 電話聯繫、Mail聯繫、邀line群組 |
| 引薦熱度 | 進度條 | 否 | 低到高（冷/溫/熱） |
| 引薦內容 | 文字輸入 | 是 | 引薦說明 |

---

### 6. 我的榮譽

查看個人成就與統計數據。

**頁面內容：**

| 區塊 | 說明 |
|------|------|
| 等級圖示 | 🏆 獎盃圖示 |
| 等級說明 | 隨著投入值而提升達成 Level 升級目標有獎勵 |
| 等級類型 | 金質（金）、優質（綠）、一般（黑） |

**統計項目（含進度條）：**

| 項目 | 說明 |
|------|------|
| 會員交流 | 交流次數統計 |
| 商學講堂 | 參與次數統計 |
| 優質引薦 | 引薦次數統計 |
| 推薦入會 | 推薦成功次數 |
| 活動參與 | 活動參與次數 |

**使用紀錄：**

| 項目 | 一年 | 全部 |
|------|------|------|
| 提供的引薦 | 數量 | 數量 |
| 收到的引薦 | 數量 | 數量 |
| 使用點數 | 數量 | 數量 |
| 收入點數 | 數量 | 數量 |

## API 呼叫

| 功能 | API | 說明 |
|------|-----|------|
| 會員搜尋 | GET /api/v1/members/search | 搜尋會員 |
| 會員詳情 | GET /api/v1/members/:id | 取得會員資料 |
| 發起交流 | POST /api/v1/meetings | 建立交流預約 |
| 課程列表 | GET /api/v1/events?type=BUSINESS_LECTURE | 取得商學講堂列表 |
| 課程報名 | POST /api/v1/events/:id/register | 報名課程 |
| 引薦列表 | GET /api/v1/referrals/mine | 取得我的引薦 |
| 發起引薦 | POST /api/v1/referrals | 建立引薦 |
| 我的榮譽 | GET /api/v1/members/me/honors | 取得榮譽統計 |
| 我的統計 | GET /api/v1/members/me/stats | 取得使用紀錄 |
